# ✅ To-Do List Web App

A dynamic and responsive To-Do List application built with HTML, CSS, and JavaScript.

## 🔥 Features
- Add and delete tasks
- Mark tasks as complete
- Stores tasks in local storage
- Clean and responsive layout

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox)
- JavaScript (ES6+)
- Local Storage
